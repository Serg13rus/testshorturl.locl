Работает на сервере под управлением Apache!

Инструкция

1. Создать на сайте дерикторию.
2. Поместить в неё все файлы кроме .htaccess. Проследить, чтобы IDE не изменила namespase.
3. В корень сайта скопировать .htaccess или внести правила в существующий .htaccess. В правилах указать правильный путь до router.php (пр.: RewriteRule .* yourdir/router.php [L])

Хранение URL организовано в файле в виде данных JSON.
Правила из .htaccess не будут затрагивать файлы и дериктории на сайте. Остальные адреса с домена будут вести к router.php.
Для лучшего роутинга можно в правилах .htaccess добавить маску.
